How to Use:
=================================================================

1. Double-click "jianyijian.exe" to run the application
2. No Python or additional dependencies required
3. Fully self-contained executable

=================================================================
System Requirements:
=================================================================

  - Windows 7/8/10/11 (64-bit)
  - No additional software installation needed
  - Minimum 200MB free disk space

=================================================================
Features:
=================================================================

  - Support 2x2, 3x3, and custom grid splitting
  - Real-time preview with rotation support
  - Batch processing (up to 10 files)
  - Multiple output formats (PNG/JPG)
  - Smart grid detection
  - Edge line detection

=================================================================
Troubleshooting:
=================================================================

If the application fails to start:

1. Check if antivirus software blocked the executable
2. Try running as administrator
3. Ensure Windows is up to date
4. Check if Visual C++ Redistributable is installed:
   https://aka.ms/vs/17/release/vc_redist.x64.exe
